## Quick Start

```bash

# 2. Install dependencies
pnpm install

# 3. Edit your component
# â†’ registry/new-york/your-component.tsx

# 4. Update registry.json with your component details

```

Once deployed, users install your component with:

```bash
npx shadcn@latest add https://your-site.vercel.app/r/your-component.json
```
